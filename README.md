# Climate CRM — Wave Exponential BCG

**Live:** https://zzjobzzz-hub.github.io/Climate_crm/

## No build required

This app uses React 18 + Babel loaded from CDN.
GitHub Actions just uploads the two files — no npm, no node.

## Files
- `index.html` — loads React + Babel from CDN, mounts the app
- `App.jsx` — full CRM application
- `.github/workflows/deploy.yml` — auto-deploys on push to main

## Enable GitHub Pages
Settings → Pages → Source → **GitHub Actions**
